# Sound Effects Directory

This directory contains audio files for the game's sound effects system.

## Required Sound Files

The following sound files are needed (in `.mp3` format):

1. **correct-guess.mp3** - Plays when a player guesses the word correctly
2. **your-turn.mp3** - Plays when it's your turn to draw
3. **round-start.mp3** - Plays at the beginning of each round
4. **timer-warning.mp3** - Plays when time is running low (≤10 seconds)
5. **game-end.mp3** - Plays when the game ends
6. **player-join.mp3** - Plays when a player joins the room
7. **player-leave.mp3** - Plays when a player leaves the room
8. **chat-message.mp3** - Plays for incoming chat messages
9. **click.mp3** - UI click sound (optional)

## How to Add Sound Files

1. Find or create audio files for each sound effect
2. Convert them to `.mp3` format (or `.ogg` for better browser support)
3. Name them exactly as listed above
4. Place them in this directory

## Recommended Sources for Free Sound Effects

- **Freesound.org** - https://freesound.org/
- **Zapsplat** - https://www.zapsplat.com/
- **Mixkit** - https://mixkit.co/free-sound-effects/
- **Pixabay** - https://pixabay.com/sound-effects/

## Sound Characteristics

For best results:
- **Duration**: Keep sounds short (0.5-2 seconds)
- **Volume**: Normalize audio levels
- **Format**: MP3 (128-192 kbps) or OGG for smaller file sizes
- **Style**: Use pleasant, non-intrusive sounds that enhance gameplay

## Testing

Once you've added the sound files, test them by:
1. Starting the development server
2. Joining a game room
3. Triggering various game events
4. Adjusting volume controls to ensure they work properly
